# How to Cite UCPIS

This document explains how to correctly cite the
**Universal Cyber-Physical Interoperability Stack (UCPIS)** in reports,
papers, policy documents, standards discussions, and derivative works.

---

## Canonical Reference

The authoritative source for UCPIS is the public GitHub repository:

https://github.com/ucpis2026us/ucpis

All official documents, annexes, release notes, and citation metadata
are maintained there.

---

## Standard Citation (Recommended)

When referencing the UCPIS architecture, cite as:

**Michael James Malecek (2026).**  
*Universal Cyber-Physical Interoperability Stack (UCPIS): Public White Paper v1.3.*  
GitHub repository: https://github.com/ucpis2026us/ucpis

## Machine-Readable Citation

This repository includes a `CITATION.cff` file that enables automatic
citation generation via GitHub’s **“Cite this repository”** feature.

Supported formats include:
- APA
- BibTeX
- RIS
- EndNote

Users are encouraged to rely on this metadata for accurate attribution.
