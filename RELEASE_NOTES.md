# UCPIS Release Notes

## UCPIS v1.3 — Public Reference Architecture

**Release Year:** 2026  
**Status:** Public, Non-Normative Reference

This release publishes the first complete public corpus of the
Universal Cyber-Physical Interoperability Stack (UCPIS).

UCPIS is an open reference architecture for interoperable cyber-physical
systems integrating AI, automation, energy, logistics, and human governance.

## Included Documents

- UCPIS White Paper v1.3
- Annex A v1.1 — Agent-Agnostic Interfaces & Human Autonomy Profiles
- Annex B v1.0 — AI as Electrical and Thermal Load
- Annex C v1.0 — Reference Architecture Diagrams
- Annex D v1.0 — Threat Model & Resilience
- Annex E v1.0 — Standards Alignment & Mapping
- Governance Without Capture

## Clarifications

- No normative security controls are defined
- No compliance or certification claims are made
- This release defines architecture only
